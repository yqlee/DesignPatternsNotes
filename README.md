##DesignPatternsNotes##
1. 简述：  
   这是一个设计模式的学习笔记！当代码写到一定量的时候，我们不禁会思考如何才能把代码写得扩展性更强，更为优雅，而不仅仅是功能实现，那么设计模式就至关重要！
